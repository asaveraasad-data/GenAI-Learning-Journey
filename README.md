# 🚀 GenAI Learning - LangChain Models

This repository documents my journey of learning Generative AI using LangChain.

## 📚 Day 4 - LangChain Models

### What I Learned

- Learned the role of the **Models** component in LangChain.
- Understood the difference between **closed-source** and **open-source** LLMs.
- Explored how LangChain provides a common interface for different LLM providers.
- Implemented examples using:
  - OpenAI
  - Claude
  - Hugging Face (Qwen2.5-7B-Instruct)
- Successfully ran my first LangChain application using an open-source model from Hugging Face.

### My Reflection

Running my first LangChain code was exciting! It was interesting to see how easily I could switch between different LLM providers using the same framework. Looking forward to learning more and building bigger projects.
